# Codex Instructions

이 프로젝트는 초보 개발자가 만드는 감정 기반 장소 추천 AI 서비스다.

## 중요한 규칙
- 코드를 수정하기 전에 먼저 수정 계획을 설명한다.
- 초보자도 이해할 수 있게 주석을 많이 단다.
- 한 번에 너무 많은 파일을 만들지 않는다.
- 실행 가능한 작은 단위로 작업한다.
- 오류가 있으면 원인 → 해결 → 수정 코드 순서로 설명한다.
- 사용자의 핵심 아이디어인 감정 8축 F/A/D/J/C/G/T/R 구조를 유지한다.
- 처음에는 복잡한 머신러닝보다 규칙 기반 점수 계산으로 시작한다.
- API 키나 비밀번호는 코드에 직접 넣지 않는다.

## 선호 구조
- backend/
- frontend/
- docs/
- data/
- tests/

## 우선순위
1. 작동하는 MVP
2. 이해하기 쉬운 코드
3. 나중에 확장 가능한 구조
