# 다모여 / Emotion Wave AI

## 핵심 아이디어
사용자의 글이나 대화를 분석해서 감정 8축을 계산하고,
그 감정 상태에 맞는 장소, 콘텐츠, 메시지를 추천하는 AI 서비스다.

## 감정 8축
- F: 공포
- A: 분노
- D: 우울
- J: 기쁨
- C: 호기심
- G: 욕심
- T: 긴장
- R: 안정

## 기본 흐름
1. 사용자가 오늘 기분이나 상황을 입력한다.
2. AI가 텍스트를 감정 8축 점수로 변환한다.
3. 장소 데이터와 감정 벡터를 비교한다.
4. 사용자에게 TOP 5 장소를 추천한다.
5. 왜 이 장소가 맞는지 설명한다.

## MVP 목표
처음부터 거대한 AI를 만들지 않는다. 먼저 다음 기능만 만든다.

- 텍스트 입력
- 감정 8축 점수 계산
- 간단한 장소 추천
- 추천 이유 출력
- FastAPI 백엔드
- 나중에 Next.js 프론트엔드 연결

## 기술 스택
- Python
- FastAPI
- PostgreSQL 또는 SQLite
- 나중에 Next.js
- 나중에 KakaoMap API 연결
